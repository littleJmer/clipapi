# Requerimientos

- docker
- docker-compose

# Despliege

Para iniciar Clip API es neceserio iniciar el siguiente bashscript que se encuentra en el root del proyecto. Este bashscript iniciara docker-compose.

```bash
sh deploy.sh
```
